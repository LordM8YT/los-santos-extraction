# extraction_pause

Custom pause shell for Los Santos Extraction.

## Purpose

- Suppresses the native GTA pause/menu flow.
- Opens an LSX pause NUI on ESC/P.
- Routes players to lobby, inventory, and raid leave actions.
- Keeps the future tactical map separate from the native GTA map.

Do not put gameplay authority in this resource. It is a client-side shell and should call existing server-validated systems.
